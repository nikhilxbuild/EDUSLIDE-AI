# **App Name**: EduSlide

## Core Features:

- PDF Upload: Allow students to upload PDF files for conversion.
- Page Selection and Reordering: Enable users to select, remove, and reorder pages using thumbnail previews.
- N-Up Layout Customization: Provide manual controls for rows (1–8) and columns (1–8). No presets.
- Layout Options: Portrait / Landscape orientation, and Default / Minimal / No margins.
- Color Mode Selection: Normal, Grayscale, B&W, and Invert.
- Auto Clean-Up: Remove blank pages and crop borders automatically.
- PDF Generation & Download: Generate and provide a downloadable optimized PDF.

## Style Guidelines:

- Clean, centered layout. Fully responsive for mobile. One consistent background across all pages. No grey/patchy sections.
- Use 'Inter' font for headings and body.
- Minimal, consistent icon set. Use accent color for active states.
- Subtle loading and button feedback only. No heavy animations.
- Background color: Dark charcoal (#0B0F1A) for a consistent dark theme.
- Card background color: Dark grey (#121826) for UI elements.
- Border color: Dark grey (#1F2937) to define UI boundaries.
- Accent color: Purple (#8B5CF6) for interactive elements and primary calls to action.
